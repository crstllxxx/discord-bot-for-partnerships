# Discord Ads Bot

Бот публикует рекламу других серверов в заданный канал по расписанию.

## Что умеет

- Отправляет объявления из `ads.json` по времени из `config.json`
- Позволяет удобно добавлять/убирать рекламу через отдельный файл
- Поддерживает timezone
- Имеет slash-команды администратора:
  - `/ads_reload` — перезагрузить `config.json` и `ads.json` без перезапуска
  - `/ads_send_now` — отправить следующее объявление сразу
  - `/ads_add` — добавить новую рекламу прямо из Discord-клиента
  - `/ads_list` — показать список реклам с номерами
  - `/ads_remove` — удалить рекламу по номеру или названию

## Установка

1. Установите Python 3.11+ (включая Python 3.14)
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

Дополнительно: для этого проекта не требуется установка Rust или Visual Studio Build Tools.

3. Создайте `.env` на основе примера:

```bash
copy .env.example .env
```

4. В `.env` укажите токен:

```env
DISCORD_TOKEN=ваш_токен_бота
```

5. Создайте `config.json` на основе примера:

```bash
copy config.example.json config.json
```

6. Заполните:
- `channel_id` — ID канала, куда слать рекламу
- `timezone` — например `Europe/Moscow`
- `post_times` — список времени в формате `HH:MM`

7. Отредактируйте `ads.json`:
- добавляйте и удаляйте объекты в массиве `ads`

## Запуск

```bash
python bot.py
```

## Пример `ads.json`

```json
{
  "ads": [
    {
      "name": "Game Hub",
      "invite_url": "https://discord.gg/yourinvite",
      "description": "Активное комьюнити, ивенты и розыгрыши"
    }
  ]
}
```

## Важно по правам

- Боту нужны права на отправку сообщений в целевой канал
- Команды управления ограничены администраторами сервера
